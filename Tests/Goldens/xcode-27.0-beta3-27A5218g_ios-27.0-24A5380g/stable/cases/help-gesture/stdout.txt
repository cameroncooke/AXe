OVERVIEW: Perform preset gesture patterns on the simulator.

Execute common gesture patterns without specifying coordinates.

Available presets:
  scroll-up, scroll-down, scroll-left, scroll-right
  swipe-from-left-edge, swipe-from-right-edge
  swipe-from-top-edge, swipe-from-bottom-edge

Examples:
  axe gesture scroll-up --udid SIMULATOR_UDID
  axe gesture scroll-down --duration 1.5 --udid SIMULATOR_UDID
  axe gesture swipe-from-left-edge --screen-width 430 --screen-height 932
--udid SIMULATOR_UDID

USAGE: axe gesture <preset> [--screen-width <screen-width>] [--screen-height <screen-height>] [--duration <duration>] [--delta <delta>] [--pre-delay <pre-delay>] [--post-delay <post-delay>] --udid <udid>

ARGUMENTS:
  <preset>                The gesture preset to perform. (values: scroll-up,
                          scroll-down, scroll-left, scroll-right,
                          swipe-from-left-edge, swipe-from-right-edge,
                          swipe-from-top-edge, swipe-from-bottom-edge)

OPTIONS:
  --screen-width <screen-width>
                          Screen width in points (default: 390 for iPhone 15).
  --screen-height <screen-height>
                          Screen height in points (default: 844 for iPhone 15).
  --duration <duration>   Duration of the gesture in seconds (uses preset
                          default if not specified).
  --delta <delta>         Distance between touch points in pixels (uses preset
                          default if not specified).
  --pre-delay <pre-delay> Delay before starting the gesture in seconds.
  --post-delay <post-delay>
                          Delay after completing the gesture in seconds.
  --udid <udid>           The UDID of the simulator.
  --version               Show the version.
  -h, --help              Show help information.
