OVERVIEW: Install AXe skill files for detected AI clients.

USAGE: axe init [--client <client>] [--dest <dest>] [--force] [--uninstall] [--print]

OPTIONS:
  --client <client>       Target client: auto, claude, or agents. Defaults to
                          auto-detect. (values: auto, claude, agents; default:
                          auto)
  --dest <dest>           Custom destination skills directory (overrides
                          --client).
  --force                 Overwrite an existing installed skill.
  --uninstall             Remove installed AXe skill from target directories.
  --print                 Print bundled skill content to stdout.
  --version               Show the version.
  -h, --help              Show help information.
